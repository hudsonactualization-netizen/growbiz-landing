How it works

This landing page is always online. When you click Open GrowBiz App, it pulls the latest live server URL from live.json and redirects you. If the server is down, you’ll get a clear status message instead of a broken experience.

FAQ

Why does the live URL change sometimes?

Early stage hosting uses a tunnel link to your server. If the tunnel restarts, the link may change. This landing page automatically fetches the latest live link.

Will my website be accessible to anyone on the internet?

Yes — as long as the server is running and the tunnel is live. If the server goes offline, pages won’t load until it’s back online.

Is GrowBiz a replacement for web agencies?

It can be. GrowBiz provides websites + daily business tools, and automates delivery — designed for speed, clarity, and low overhead.
